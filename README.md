# SmartDoc AI

An AI-powered document intelligence platform that allows users to upload PDF documents and ask natural language questions using Retrieval-Augmented Generation (RAG).

## Features

- PDF Upload
- AI Question Answering
- Context Retrieval
- Conversation History
- Fast LLM Responses
- Groq Llama Integration

## Tech Stack

- Python
- Gradio
- LangChain
- ChromaDB
- HuggingFace Embeddings
- Groq API
